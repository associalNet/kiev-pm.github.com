Что

Perl Ski Hackathon 2013 это попытка провести зимний Perl хакатон на Украине. Хакатон это событие, когда программисты собираются вместе для написания чего-то полезного (или нет), для социализации и чтобы повеселиться.

Хотя темы могут варьироваться, обычно формируются несколько групп по интересам. Список предложенных тем будет анонсирован позже.
